## Skill: Family Subscription Summary

Use this skill when the user wants a family subscription list or asks to compare recurring packages.

Required response shape:
- Show `งวดของการชำระ`
- Show `user owner`
- Show `name`
- Show `value`

Guidance:
- Keep the result compact and easy to scan.
- Prefer a table when multiple subscriptions are present.
- If the user asks about changes, confirm which subscription should be updated first.
