## Skill: True Mobile Promotion Finder

Use this skill when the user asks about the current promotion on a True Mobile package.

Required response shape:
- State the current package
- State the promotion name
- State the discount or benefit
- State the eligibility window or condition

Guidance:
- Compare the current package to the available promotion.
- Mention if the promotion applies only to existing family customers or specific plan types.
- Keep the explanation direct and action-oriented.
