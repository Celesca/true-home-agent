## Skill: IoT Household Device Control

Use this skill when the user wants to shut down, pause, or manage household devices like TV, tablet, speaker, or other smart devices.

Required response shape:
- Confirm the target device name
- Confirm whether it is a one-time action or a schedule
- State the resulting device status

Guidance:
- Ask one clarifying question if the device name is ambiguous.
- Prefer a safe pause action over destructive wording.
- If the request mentions TV or similar, map it to the matching household device list.
