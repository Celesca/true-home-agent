## Skill: TrueWiFi Router Support

Use this skill when the user wants to adjust Wi-Fi speed, check router status, or diagnose router problems.

Required response shape:
- State the current router or network status
- State the likely cause
- State the next action
- If relevant, note the current speed profile

Guidance:
- Separate service-side issues from coverage or device-side issues.
- If the user asks to change speed, confirm the target speed profile before applying it.
- Keep diagnostics short and useful.
